Oblog Documentation
===================

Welcome to the documentation for Oblog, an annotation-based `Shuffleboard <https://docs.wpilib.org/en/latest/docs/software/wpilib-tools/shuffleboard/getting-started/shuffleboard-tour.html>`__ logging tool for FRC teams!

For an overview of what Oblog is and why you should use it, see the :ref:`introduction page <introduction:Introduction to Oblog>`.  For a quick start guide, see :ref:`getting-started:Getting Started with Oblog`.

The Oblog API docs can be found `here <https://oblarg.github.io/Oblog/>`__.

An example project (based on the WPILib FrisbeeBot example project) demonstrating the usage of several Oblog features can be found `here <https://github.com/Oblarg/Oblog/tree/master/examples/src/main/java/io/github/oblarg/logexample>`__.
